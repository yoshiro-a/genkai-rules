# genkai-rules
ゲーム限界都市のルールをバージョン管理、公開するためのリポジトリ

## 公開先
https://diy-gifu.github.io/genkai-rules

## 概要
- Web版と印刷版の両方のルールが見れるようにします
- `master`ブランチの`docs`内がGitHub Pagesで公開されています
- ドキュメンテーションサイトが作れるdocsifyを使用しています

## 編集方法
### GitHubから
1. docs内の編集したいMarkdownファイルを開きます
2. 右上の✏をクリックし、編集します
3. 新規ブランチを作成しコミットします
4. プルリクエストを出します
